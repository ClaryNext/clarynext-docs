# Activity Settings

Access settings by clicking **Settings** in the designer toolbar.

## General Settings

- **Title** — Activity name shown to users
- **Description** — What the activity helps achieve
- **Category** — For marketplace organization
- **Tags** — Keywords for searchability
- **Thumbnail** — Preview image

## Engagement Settings

### Mode

- **On-Demand** — Run whenever users choose
- **Scheduled** — Set times with reminders
- **Expert-Guided** — With mentor support

### Repetition

- **Allow Repeat Runs** — Multiple engagements
- **Track History** — Save all run data

## Visibility Settings

- **Private** — Only you can access
- **Shared** — Specific users or groups
- **Organization** — All org members
- **Marketplace** — Published for purchase

## Data Settings

- **Response Storage** — All user inputs saved
- **Privacy** — Control who sees responses
