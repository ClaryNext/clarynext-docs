# Sharing & Publishing

## Sharing Options

| Option | Who Can Access |
|--------|---------------|
| **Private** | Only you |
| **Direct Share** | Specific individuals |
| **Organization** | Org members |
| **Marketplace** | Anyone |

## Direct Sharing

1. Open activity in designer
2. Click **Share**
3. Enter email address(es)
4. Set permissions (View, Edit, Admin)
5. Send invitation

## Organization Sharing

1. Open activity settings
2. Set visibility to **Organization**
3. Choose specific teams or all members

## Publishing to Marketplace

### Process

1. Complete and test your activity
2. Add thumbnail and description
3. Set pricing (Free, One-Time, or Credits)
4. Click **Publish**
5. Activity appears in marketplace

### Pricing Options

- **Free** — No charge
- **One-Time Purchase** — Single payment
- **Credits** — Pay with activity credits

## Managing Published Activities

- Push updates to all users
- View analytics and reviews
- Unpublish if needed
