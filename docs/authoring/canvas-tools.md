# Canvas Tools

The canvas is where you visually design your activity flow.

## Navigation

- **Pan** — Click and drag on empty space
- **Zoom** — Mouse wheel or pinch gesture
- **Fit to View** — Click fit button in toolbar

## Working with Nodes

### Adding Nodes

- Drag components from the left panel to canvas
- Or right-click on canvas and select "Add Step"

### Selecting and Moving

- Click to select a node
- Shift+click for multi-select
- Drag to move nodes

### Deleting

- Select node and press Delete
- Or right-click and select "Delete"

## Connections

### Creating Connections

1. Hover over a node's output handle (bottom)
2. Click and drag to target node's input handle (top)
3. Release to create connection

### Deleting Connections

- Click connection to select
- Press Delete

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Undo | Ctrl/Cmd + Z |
| Redo | Ctrl/Cmd + Shift + Z |
| Copy | Ctrl/Cmd + C |
| Paste | Ctrl/Cmd + V |
| Delete | Delete/Backspace |
| Save | Ctrl/Cmd + S |
