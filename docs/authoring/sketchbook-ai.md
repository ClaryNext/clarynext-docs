# Sketchbook AI

Sketchbook AI helps you create activities using artificial intelligence.

## Getting Started

1. Go to **Design** in navigation
2. Click **Create with AI** or **Sketchbook AI**

## Creating Activities

### From Description

1. Click **New Activity from Description**
2. Describe what you want to create
3. Click **Generate**
4. Review and refine the result

### From Outline

1. Click **New Activity from Outline**
2. Enter your step-by-step outline
3. Click **Generate**
4. AI fills in each step with appropriate components

## Refining Output

After generation:
- Review each step
- Edit content directly
- Request changes: "Make this more concise"
- Regenerate sections if needed

## AI Credits

AI generation consumes credits:
- View balance at **Account > Credits**
- Credits refresh with subscription
- Additional credits can be purchased
