# Text Components

Text components display content to users.

## Rich Text

The most versatile display component.

### Features

- Headings (H1-H6)
- Paragraphs and formatting (bold, italic)
- Bulleted and numbered lists
- Links (internal and external)
- Inline images
- Callout boxes (tips, warnings, notes)

### Adding Rich Text

1. Drag Rich Text from component panel
2. Click to edit content
3. Use toolbar for formatting

## Image Component

Display standalone images.

1. Drag Image component
2. Upload image or enter URL
3. Add alt text for accessibility
4. Configure size and alignment

Supported: JPG, PNG, GIF, SVG, WebP

## Video Component

Embed video content.

### Sources

- Upload directly to ClaryNext
- YouTube, Vimeo, Loom embeds

### Options

- Autoplay, controls, loop
- Start/end time
- Optional transcript

## Separator

Visual breaks between sections:
- Line separators
- Space separators
