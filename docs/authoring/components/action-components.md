# Action Components

Action components create tasks and follow-ups that appear in the user's Workspace.

## Task Component

Create actionable to-do items.

### How It Works

1. User encounters task in activity
2. Reviews task description
3. Sets optional due date
4. Task appears in Workspace
5. User completes and marks done

### Configuration

- **Title** — Task name
- **Description** — What to do
- **Due Date** — Suggested deadline
- **Priority** — High, Medium, Low

## Reminder Component

Schedule future notifications.

### Configuration

- **Message** — Reminder content
- **Timing Options** — When to remind
- **Repeat** — Recurring reminders
- **Delivery** — Email, push, in-app

## Commitment Component

Capture personal pledges.

- User articulates commitment
- Adds specifics and timeline
- Tracked in Workspace

## Action Item Generator

Automatically create action items from user responses.

Can use AI to suggest relevant action items based on activity inputs.
