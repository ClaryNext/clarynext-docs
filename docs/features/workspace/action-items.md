## Creating Items from Activities

During any activity:
1. Click **â‹®** on a text field
2. Select **Add as action item**
3. Item appears in Take-Aways panel and later in Workspace

## Working with Sub-tasks

Large action items can be broken into smaller steps:
1. Select an action item
2. Click **Add Sub-tasks** or use **Break it down**
3. Sub-tasks appear nested under the parent
4. Complete sub-tasks to make progress on the main item
