# Reminders

Reminders keep you engaged with activities and action items.

## Types of Reminders

| Type | Purpose |
|------|---------|
| **Activity Reminders** | Return to scheduled activities |
| **Action Item Reminders** | Complete pending tasks |
| **Custom Reminders** | Personal notifications |

## Managing Reminders

### From Workspace

1. Navigate to **Workspace**
2. Click **Reminders** tab
3. View and manage all reminders

### Creating Custom Reminders

1. Click **New Reminder**
2. Enter content
3. Set date and time
4. Configure repeat schedule (optional)

## Delivery Methods

- **In-App** — Notifications in ClaryNext
- **Email** — Individual or digest emails
- **Push** — Browser notifications

## Settings

Access notification preferences in **Settings > Notifications**:
- Quiet hours
- Delivery preferences
- Frequency limits
