# Activity Components

Activities are built from modular components.

## Component Categories

| Category | Purpose | Examples |
|----------|---------|----------|
| **Display** | Show content | Text, Video, Image |
| **Input** | Collect responses | Text fields, Sliders, Ratings |
| **Action** | Prompt tasks | Tasks, Reminders |
| **AI** | Intelligent feedback | AI Prompts |

## Display Components

- **Rich Text** — Formatted text, headings, lists
- **Video** — Embedded video content
- **Image** — Images and galleries

## Input Components

- **Short Text** — Single-line text entry
- **Long Text** — Multi-line responses
- **Slider** — Visual range selection
- **Rating** — Star-based evaluation
- **Single Choice** — Select one option
- **Multiple Choice** — Select multiple options
- **Date/Time Picker** — Date and time selection

## Action Components

- **Task** — Actionable to-do items saved to Workspace
- **Reminder** — Scheduled notifications

## AI Components

- **AI Prompt** — Context-aware AI responses and feedback
