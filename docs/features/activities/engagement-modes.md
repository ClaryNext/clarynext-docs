# Engagement Modes

Activities support different engagement modes.

## Overview

| Mode | Description |
|------|-------------|
| **On-Demand** | Run whenever you choose |
| **Scheduled** | Runs at set times with reminders |
| **Expert-Guided** | Run with mentor support |

## On-Demand Mode

The most flexible mode — run activities whenever needed.

1. Access from dashboard or marketplace
2. Start when ready
3. Complete at your own pace
4. Pause and resume as needed

## Scheduled Mode

Activities with regular schedules and reminders.

- **Daily** — Once per day
- **Weekly** — Specific days
- **Monthly** — Specific dates

You receive reminders and can track completion consistency.

## Expert-Guided Mode

Activities run with mentor support.

1. Schedule a session with your mentor
2. Both join at the scheduled time
3. Mentor sees your progress and provides guidance
4. Real-time discussion and feedback
