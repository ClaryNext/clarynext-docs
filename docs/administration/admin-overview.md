# Administration Overview

As a ClaryNext administrator, you manage users, configure the platform, and ensure organizational success. This guide introduces administrative capabilities.

## Administrator Roles

### System Administrator

Full platform control:
- User management
- Organization configuration
- Marketplace settings
- AI configuration
- All administrative functions

### Organization Administrator

Organization-level control:
- Manage organization users
- Configure organization settings
- Access organization analytics
- Manage organization content

### Team Administrator

Team-level control:
- Manage team members
- Assign activities
- View team progress
- Limited configuration

## Accessing Administration

![Admin Panel](/img/screenshots/admin-panel.gif)

### Admin Dashboard

1. Log in with admin account
2. Click **Admin** in navigation
3. Access administrative features

### Admin Features

The admin dashboard provides:
- **Users** — User management
- **Organizations** — Org configuration
- **Content** — Activity management
- **Marketplace** — Commerce settings
- **AI** — AI configuration
- **Analytics** — Platform insights
- **Settings** — System settings

## What Administrators Can Do

### User Management

- Create and manage user accounts
- Assign roles and permissions
- Reset passwords
- Enable/disable accounts
- View user activity

### Organization Setup

- Create organizations
- Configure organization settings
- Manage organization hierarchy
- Set organization branding
- Assign organization admins

### Content Management

- Review submitted activities
- Approve marketplace listings
- Feature content
- Remove policy violations
- Manage categories and tags

### Marketplace Administration

- Configure pricing rules
- Manage vendor accounts
- Review refund requests
- Set featured products
- Configure payment settings

### AI Configuration

- Configure AI prompts
- Manage AI examples
- Set up knowledge base
- Monitor AI usage
- Adjust AI settings

### Analytics & Reporting

- View platform analytics
- Generate usage reports
- Monitor health metrics
- Track growth indicators
- Export data

## Administrative Best Practices

### Security

- Use strong passwords
- Enable two-factor auth
- Limit admin access
- Audit admin actions
- Regular access reviews

### User Management

- Onboard users systematically
- Clear role definitions
- Regular permission reviews
- Quick issue resolution
- Clear communication

### Content Quality

- Clear content guidelines
- Consistent review process
- Fair enforcement
- Quality standards
- Regular audits

### Platform Health

- Monitor performance
- Regular maintenance
- User feedback review
- Proactive improvements
- Documentation updates
