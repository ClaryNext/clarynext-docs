# Mentor Sessions

Mentor sessions allow you to guide mentees through activities in real-time.

## Session Types

- **Live Guided** — Real-time walkthrough together
- **Async Review** — Review and comment after completion
- **Check-In** — Brief progress review

## Starting a Session

### Scheduled

1. Go to **Sessions** or **Calendar**
2. Click on scheduled session
3. Click **Start Session**

### Ad-Hoc

1. Open mentee profile
2. Click **Start Session**
3. Select activity

## Session Interface

As mentor, you see:
- Activity canvas (same as mentee)
- Mentor panel with tools
- Progress indicator
- Chat/communication

### Mentor Tools

- Navigation controls
- Note taking (private and shared)
- Text chat

## After the Session

- Write session summary
- Create follow-up action items
- Complete documentation

## Scheduling Sessions

1. Go to **Calendar**
2. Click **New Session**
3. Select mentee
4. Choose date/time and activity
5. Send invitation
