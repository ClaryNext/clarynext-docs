# Groups & Organizations

## Groups

Groups allow you to organize mentees into cohorts.

### Creating Groups

1. Go to **Mentees > Groups**
2. Click **New Group**
3. Enter name and description
4. Add members

### Group Activities

Assign activities to entire groups:
1. Open group
2. Go to **Activities**
3. Click **Assign Activity**

### Group Sessions

Run sessions with multiple mentees together.

## Organizations

If you're part of an organization:

- Access organization activities
- See assigned mentees
- Follow organization policies
- Report to org admins

## Collections

For informal organization:
- Filter your mentee list
- Batch operations
- Personal organization
